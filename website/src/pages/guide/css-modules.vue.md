nut 默认支持 CSS Modules

使用 `.module.[extension]` 后缀，即可开启 CSS Modules 特性

## 举例

| 后缀    | css modules 后缀 |
| ------- | ----------------:|
| .css    |      .module.css |
| .less   |     .module.less |
| .scss   |     .module.scss |
| .sass   |     .module.sass |
| .styl   |     .module.styl |
| .stylus |   .module.stylus |
