# sao-sample

> my amazing SAO generator

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao sample my-project
```

### From git

```bash
sao centmaster/sao-sample my-project
```
